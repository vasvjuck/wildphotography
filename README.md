
# WILDPHOTOGRAPHY

- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [styled-components](https://styled-components.com/)

## Running Locally

```bash
git clone https://github.com/vasvjuck/wildphotography.git
cd wildphotography
yarn install
yarn run dev
```
